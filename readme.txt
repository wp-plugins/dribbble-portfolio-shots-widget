Dribbble Portfolio Shots Widget is a Wordpress Plugin for displaying Dribbble Portfolio items on wordpress widget positions.

== Description ==

Dribbble becoming popular day by day for designers and designers submit their portfolio items on it. We can understand the importance of Portfolio and Dribbble Portfolio Shots Widget will allow developer to put designer portfolio on their website widget positions.

Few Features you will love

*   Set default values to give you easy start
*   Responsive in Nature
*   Available 2 column, 4 column, 6 column grid

== Installation ==

Installing this plugin is same like other wordpress plugins

1. Upload `dribbble-portfolio-shots-widget` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the widget on Widget area of wordpress by dragging and drop the item.

You can install the zip file as well

Configurations:
1. Title: Title of the widget
2. Dribbble Username: Your Dribbble Username - Example: "dribbble"
3. No. of Columns: Three Options available - "2", "4" & "6"

More info -> http://ramit-designs.com/wordpress/
